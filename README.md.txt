# TLB Visualizer Pro

An interactive web-based simulator for understanding
Translation Lookaside Buffers (TLB) in Operating Systems.

## Features
- TLB Hit/Miss Visualization
- LRU, FIFO, Random Replacement Policies
- Real-time EMAT Calculation
- Page Table Simulation
- TLB Flush Simulation
- Step-by-Step Execution Mode

## Technologies Used
- HTML5
- CSS3
- JavaScript
- SVG Animations
